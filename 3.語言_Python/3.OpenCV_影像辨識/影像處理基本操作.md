
### 讀取 cv2.imread(filename[,fils])
```
pic = cv2.imread("human12.jpg",-1)
print(pic) // 寫出
```
裡面放完整檔案名(包含副檔名)

### 顯示

#### nameWindow
建立視窗
```
Node = cv2.namedWindow("視窗名稱") # 
```
#### imshow()
顯示影像
```
pic = cv2.imread("human_face.jpg")
cv2.namedWindow("lesson")
cv2.imshow("lesson",pic) # 顯示
```
