
### Math.Max(x,y)
find the highest value of x and y
```
Math.Max(12,10);
```

### Math.Sqrt(x)
returns the square root of x
```
Math.Sqrt(64); // 回傳 double
```

### Math.Abs(x)
絕對值
returns the absolute (positive) value of x
```
Math.Abs(-4.7); // 4.7
```

### Math.Round(x)
四捨五入
rounds a number to the nearest whole number
```
Math.Round(9.99); // 10
```
