
#### System.String
來自 .NET Framework System.String 類別。

#### 參考型別
字串為參考型別。
意味 new 之後，每次的修改值其實是重新指定字串。

```
string str = "ABC";
Write(str[0]); // A
```
#### 比較運算子
雖然 string 資料型別是參考型別，可用 比較運算子(「==」和「!=」)。

#### []
如同一維陣列取值。

#### 字串插值 String.Format
```
string str = String.Format("姓名{0}，電話{1}。","Jay","321542");
```
