### 副檔名 : cs

### 主控台應用程式
進入點為 Program.cs
Program.cs 為預設名稱。
擁有一個 Main()

### Window應用程式
除了 Program.cs
另有 Forml.cs(預設名稱) : 為一圖形使用者介面的類別檔。

---

## 結構

### using
匯入類別。
```
using System;
// 匯入所指用的 Net Framework 類別。
```

### namespace
建立 class群組，避免匯入類別時名稱衝突。

### 宣告類別


# Statements
C# 必須以" ; "結尾。

# Blocks
程式區塊。
以 { } 包著。
