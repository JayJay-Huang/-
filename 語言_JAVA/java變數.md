>宣告變數，即是 `分配記憶體`。並將其 `命名`。



---
## Java `一定要指定型態`
---

    int i, j ,k ; // 宣告多個變數
    int i = 1; // 宣告給值

* 要指定 `資料型態`。
* 要「`;`」號結尾。
* `大小寫` 有差別。
* 其具有 `唯一性`，相同範圍不能重複宣告。

---
## `識別字` 慣用命名方式 : 
---

* 常數 : 大寫和「`_`」符號。
* 變數 : 小寫駝峰。
* 類別 : 大寫駝峰。
* 函數 : 小寫駝峰。

---
## `final` > 常數(Constant)
---

>變數前面加上「`final`」成為常數。

具有永不改變特性，只能指派一次。
