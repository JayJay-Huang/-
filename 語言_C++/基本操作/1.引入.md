### header file library: #include
input and output objects
意旨:使用 函式庫
```
#include <stdio.h>
// 
#include <iostream>
// 負責輸入 輸出
### #include <string>
```
### namespace: using
use names for objects and variables from the standard library
意旨:使用 標準庫
(C++標準庫 定義名稱都定義於namespace，避免名稱衝突)
引用方式
```
using namespace std;
或
std::
```
