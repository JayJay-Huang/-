## 事件驅動程序

---
Node.js 使用事件驅動模型。

1. 當web server接收到請求，就把它關閉然後進行處理，然後去服務下一個web請求。
2. 當這個請求完成，它被放回處理隊列，當到達隊列開頭，這個結果被返回給用戶。

webserver一直接受請求而不等待任何讀寫操作。（這也被稱之為非阻塞式IO或者事件驅動IO）
