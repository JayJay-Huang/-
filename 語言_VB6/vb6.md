# VB簡介
Visual Basic
微軟公司開發的包含環境的事件驅動程式語言。
源自於BASIC程式語言。

VB擁有圖形化使用者介面（GUI）和快速應用程式開發（RAD）系統，
可以輕易的使用DAO、RDO、ADO連接資料庫，或者輕鬆的建立ActiveX控制項。
