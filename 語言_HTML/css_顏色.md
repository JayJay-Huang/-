# background-color
背景顏色。
```
background-color: #6fbb9a;
```

# background-repeat
repeat: 重複
repeat-x: 水平重複
repeat-y: 垂直重複
no-repeat: 不重複

# background-position
background-repeat: no-repeat;
後可用 background-position
改變 背景圖顯示位置。

# background-size
顯示圖片尺寸。
```
background-size: 50%;
```

# color
文字顏色(前景色)。
```
color: #6fbb9a;
```
